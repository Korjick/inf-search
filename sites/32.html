    Microsoft добавила в «Проводник» рекомендации по файлам

    MicrosoftMicrosoft Windows 11
    20:00Шамиль Алиуллов24

    В тестовых сборках Windows 11 у «Проводника» появилась новая функция. Она показывает файлы, которые пользователь может захотеть открыть. В своём блоге Microsoft продемонстрировала работу этого и других нововведений из грядущего обновления. 

    Обновление появилось в сборке Windows 11 Insider Preview Build 23403, доступной через канал для разработчиков. Функция находится в тестовом режиме, поэтому доступ к ней открыли лишь некоторым тестерам. Суть нововведения заключается в отображении на главной странице «Проводника» релевантных файлов, которые пользователь, по мнению системы, захочет открыть при запуске файлового менеджера.

    В Microsoft не уточнили, на основе каких данных алгоритмы предугадывают действия пользователей, но рассказали, что функция доступна только тем, кто авторизовался в Windows с помощью учётной записи Azure Active Directory (бизнес-аккаунт). Новая вкладка «Проводника» рекомендует только облачные файлы, которые размещены в связанном с системой онлайн-хранилище. Также она может показывать чужие файлы, которыми кто-то поделился с пользователем.

    Другие нововведения сборки 23403 включают поддержку автоматических субтитров на большем количестве языков и горячие клавиши в «Проводнике».
